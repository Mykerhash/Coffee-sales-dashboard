# Coffee Sales Dashboard

## Overview
This project analyzes coffee sales data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Dashboard Design

## Key Insights
- Sales performance by country
- Top 5 customers by revenue
- Product sales trends
